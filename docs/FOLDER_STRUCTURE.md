# Folder Structure Specification

This document maps out the folder structure for the SayDump monorepo.

## Structure Overview
Refer to the main `README.md` or this specification for exact paths:

- `client/` - React frontend application.
- `server/` - Node.js Express backend application.
- `docs/` - Project documentation files.
- Root files (`package.json`, `.gitignore`, `docker-compose.yml`) control workspaces, packages, and running processes.
